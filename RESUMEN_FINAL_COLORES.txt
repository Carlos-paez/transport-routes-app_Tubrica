═══════════════════════════════════════════════════════════════════════════════
  📊 COMPORTAMIENTO DEL SISTEMA
═══════════════════════════════════════════════════════════════════════════════

ORDEN DE ASIGNACIÓN DE COLORES:
1. Rutas OMEGA (al cargar la aplicación)
2. Rutas cargadas desde la base de datos (usan su color guardado)
3. Nuevas rutas creadas por el usuario (siguiente color disponible)

CONTADOR DE COLORES:
- Se reinicia al recargar la página
- Comienza en 0
- Se incrementa con cada nueva ruta
- Usa módulo (%) para ciclar después de 20 colores

PERSISTENCIA:
- Rutas OMEGA: No se guardan (se regeneran al cargar)
- Rutas de usuario: Color guardado en base de datos
- Al cargar: Rutas usan su color guardado
