  🎨 RUTAS OMEGA CON COLORES DIFERENTES
═══════════════════════════════════════════════════════════════════════════════

CAMBIO IMPLEMENTADO:
Las rutas OMEGA (rutas predefinidas de transporte) ahora se muestran cada una
con un color diferente, facilitando su identificación visual en el mapa.


═══════════════════════════════════════════════════════════════════════════════
  ANTES vs DESPUÉS
═══════════════════════════════════════════════════════════════════════════════

ANTES:
------
❌ Todas las rutas OMEGA eran color naranja (#e67e22)
❌ Difícil distinguir entre las diferentes rutas OMEGA
❌ Confusión visual cuando se muestran múltiples rutas

DESPUÉS:
--------
✅ Cada ruta OMEGA tiene un color único de la paleta
✅ Fácil identificación visual de cada ruta
✅ Mejor experiencia de usuario
✅ Colores vibrantes y diferenciados


═══════════════════════════════════════════════════════════════════════════════
  CÓMO FUNCIONA
═══════════════════════════════════════════════════════════════════════════════

1. Al cargar la aplicación, se procesan las rutas OMEGA
2. Cada ruta recibe el siguiente color de la paleta de 20 colores
3. Las rutas se trazan automáticamente con sus colores únicos
4. Los colores se asignan en el orden en que aparecen en OMEGA_WAYPOINTS

  CÓDIGO MODIFICADO
═══════════════════════════════════════════════════════════════════════════════

ANTES:
------
const layer = renderRoute(polyCoords, routeData.name, `omega-${i}`, null, true);
                                                                          ^^^^

                                                                    (todas naranjas)

DESPUÉS:
--------
const omegaColor = getNextRouteColor();
const layer = renderRoute(polyCoords, routeData.name, `omega-${i}`, null, false, omegaColor);
                                                                          ^^^^^  ^^^^^^^^^^^
                                                                    isOmega = false
                                                                    color único

═══════════════════════════════════════════════════════════════════════════════
  DIFERENCIAS ENTRE TIPOS DE RUTAS
═══════════════════════════════════════════════════════════════════════════════

RUTAS OMEGA (Predefinidas):
✓ Se cargan automáticamente al iniciar
✓ Cada una con color diferente de la paleta
✓ No se guardan en la base de datos
✓ Se pueden mostrar/ocultar desde el panel

RUTAS MANUALES (Creadas por el usuario):
✓ Se crean con trazado manual
✓ Cada una con color diferente de la paleta
✓ Se guardan en la base de datos
✓ Color persistente entre sesiones

RUTAS INTELIGENTES (OSRM):
✓ Se crean con enrutamiento automático
✓ Cada una con color diferente de la paleta
✓ Se guardan en la base de datos
✓ Color persistente entre sesiones


═══════════════════════════════════════════════════════════════════════════════
  NOTAS TÉCNICAS
═══════════════════════════════════════════════════════════════════════════════

• Las rutas OMEGA se procesan en orden secuencial
• Cada ruta obtiene el siguiente color disponible
• El contador de colores es compartido con las rutas del usuario
• Si creas rutas de usuario antes de cargar OMEGA, los colores se mezclan
• Los colores se asignan en el orden de carga


═══════════════════════════════════════════════════════════════════════════════
  PERSONALIZACIÓN
═══════════════════════════════════════════════════════════════════════════════

Si deseas cambiar los colores de las rutas OMEGA:

1. Edita el array ROUTE_COLORS en app.js
2. Agrega, elimina o reordena colores según prefieras
3. Los cambios se aplicarán automáticamente

Ejemplo:
const ROUTE_COLORS = [
  "#FF0000", // Rojo brillante
  "#00FF00", // Verde brillante
  "#0000FF", // Azul brillante
  // ... más colores
];
