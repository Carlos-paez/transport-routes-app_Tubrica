  🎨 SISTEMA DE COLORES AUTOMÁTICOS PARA RUTAS
═══════════════════════════════════════════════════════════════════════════════

CAMBIO IMPLEMENTADO:
Ahora cada ruta trazada (manual o automática) se muestra con un color diferente,
facilitando la distinción visual entre múltiples rutas en el mapa.


═══════════════════════════════════════════════════════════════════════════════
  ANTES vs DESPUÉS
═══════════════════════════════════════════════════════════════════════════════

ANTES:
------
❌ Todas las rutas tenían el mismo color azul (#3498db)
❌ Difícil distinguir entre múltiples rutas
❌ Confusión visual cuando hay muchas rutas

DESPUÉS:
--------
✅ Cada ruta tiene un color único de una paleta de 20 colores
✅ Fácil identificación visual de cada ruta
✅ Mejor experiencia de usuario
✅ Las rutas OMEGA mantienen su color naranja característico


═══════════════════════════════════════════════════════════════════════════════
  CÓMO FUNCIONA
═══════════════════════════════════════════════════════════════════════════════

1. Al crear una ruta nueva, se asigna automáticamente el siguiente color
2. Los colores se asignan en secuencia de una paleta de 20 colores
3. Después de usar los 20 colores, el ciclo vuelve a empezar
4. Las rutas OMEGA siempre mantienen el color naranja (#e67e22)

  CÓMO PROBAR
═══════════════════════════════════════════════════════════════════════════════

1. Inicia la aplicación (JavaScript o PHP)
2. Crea varias rutas usando cualquier método:
   - Trazado manual
   - Ruta inteligente
3. Observa que cada ruta tiene un color diferente
4. Las rutas OMEGA siguen siendo naranjas


═══════════════════════════════════════════════════════════════════════════════
  PERSONALIZACIÓN
═══════════════════════════════════════════════════════════════════════════════

Para cambiar los colores, edita el array ROUTE_COLORS en app.js:

const ROUTE_COLORS = [
  "#e74c3c", // Rojo
  "#3498db", // Azul
  "#TU_COLOR_AQUI", // Agrega más colores
];

Puedes:
- Agregar más colores al array
- Cambiar los colores existentes
- Reordenar los colores


═══════════════════════════════════════════════════════════════════════════════
  NOTAS TÉCNICAS
═══════════════════════════════════════════════════════════════════════════════

• El contador de colores (colorIndex) se reinicia al recargar la página
• Cada versión (JS y PHP) tiene su propio contador independiente
• Los colores se asignan en el momento de crear la ruta
• Las rutas guardadas mantienen su color al recargar
• El sistema usa operador módulo (%) para ciclar los colores
